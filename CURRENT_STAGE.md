# Current Stage

Notsy is an early full-stack study assistant prototype. It has a React/Vite frontend, an Express/MongoDB application API, a Django AI/RAG service using Gemini/Pinecone-style retrieval, and Docker Compose orchestration for local and production-like runs.

The core product shape is present: users can authenticate, create notebooks and topics, upload PDFs or YouTube resources, chat over topic resources, generate revision notes/flashcards, and view knowledge graphs. The app is not yet production-ready: service contracts, access control, AI reliability, test coverage, observability, and deployment hardening need focused stabilization.
